How It Works

Complete Flow:

  1. Instructor: Opens Stream Mode and selects a class
  2. OBS Setup: Connects to OBS Studio v31+ via WebSocket
  3. RTMP Stream: OBS streams to rtmp://198.211.107.134/live/<classId>
  4. HLS Conversion: Server transcodes RTMP to HLS chunks
  5. Student Access: Students visit viewer with ?classId=<classId>
  6. Auto-Connect: Viewer automatically detects and plays HLS stream
  7. Real-time Updates: Socket.IO handles viewer tracking and notifications
Technical Details

The system uses the following technology stack:

OBS Studio → RTMP → DigitalOcean Server → HLS → Student Browser

Server: 198.211.107.134 (DigitalOcean)

RTMP Endpoint: rtmp://198.211.107.134/live/

HLS Endpoint: http://198.211.107.134/hls/
